Why This Extension?

USA Ultimate made the controversial decision to change "Portland Rhino Slam!" to just "Portland Rhino" because someone complained about an Urban Dictionary definition. This extension fixes that absurdity and adds educational hover cards for ALL team names.

üì¶ Installation

Easy install! Download the signed extension and drag it to Chrome:

  1. Download Extension (.crx)
  2. Open Chrome and go to chrome://extensions/
  3. Enable "Developer mode" (toggle in top right corner)
  4. Drag the downloaded .crx file onto the extensions page
  5. Click "Add Extension" when prompted
  6. Visit USA Ultimate and enjoy!

Alternative: If the .crx doesn't work, download the ZIP, extract it, and load it as an unpacked extension.
