USAU Team Name Fixer

USA Ultimate changed "Portland Rhino Slam!" to "Portland Rhino" because someone complained about the Urban Dictionary definition. This Chrome extension fixes that and adds hover cards showing Urban Dictionary definitions for all team names.

Download Extension

To install:

  1. Download and extract the ZIP file
  2. Open Chrome and go to chrome://extensions/
  3. Enable "Developer mode" (toggle in top right)
  4. Click "Load unpacked"
  5. Select the extracted extension folder
